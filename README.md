# Campus Life Super App

## Description
This is a web application designed to help Millsaps College students navigate campus life. 
The app brings together dining information, student clubs, and campus events in one place.

## Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- OpenWeatherMap API (weather on home page)
- Millsaps Localist Calendar API (events page)

## Setup and Deployment
1. Clone the repository: `git clone https://github.com/caleighchapman/CampusLifeSuperApp.git`
2. Open the project folder in VS Code
3. Open `index.html` with Live Server to run locally
4. Deployed live at: YOUR_GITHUB_PAGES_URL_HERE

## Features
- **Home** - Welcome dashboard with live weather for Jackson, MS
- **Dining** - Browse Millsaps dining locations (Reuben's, The Caf, EcoGrounds) with links to real menus
- **Clubs** - Browse and filter student organizations by category
- **Events** - View upcoming Millsaps campus events pulled from the live Millsaps calendar

## How To Use
1. Use the navbar at the top to navigate between pages
2. On the Dining page, click a location button to see details and menu links
3. On the Clubs page, click a category button to filter clubs
4. On the Events page, events load automatically and link to the full Millsaps calendar
